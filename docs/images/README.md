# Screenshots Directory

This directory contains screenshots for the README.md file.

## Required Images

Please add the following screenshots to this directory:

1. **lesson-planner.png** - Screenshot of the Lesson Planner feature
2. **classrooms.png** - Screenshot of the Classrooms feature
3. **grades.png** - Screenshot of the Grades feature
4. **worksheets.png** - Screenshot of the Differentiated Worksheets feature
5. **visual-aids.png** - Screenshot of the Visual Aids feature
6. **knowledge-base.png** - Screenshot of the Knowledge Base feature
7. **local-content.png** - Screenshot of the Local Content feature
8. **teacher-companion.png** - Screenshot of the Teacher Companion feature

## Image Guidelines

- **Format**: PNG or JPG
- **Recommended size**: 1200x800 pixels or similar aspect ratio
- **Quality**: High resolution, clear and readable
- **Content**: Show the feature in action with real data (if possible)

## How to Add Images

1. Take screenshots of each feature in your application
2. Save them with the exact filenames listed above
3. Place them in this `docs/images/` directory
4. The README.md will automatically display them
